# Openbox

1. Para abrir algo no open box aperte Alt + space no teclado e digite o que desejar.

1. Para mover as telas para direita e esquerda aperte a tecla Windows + seta direita/esquerda.

1. Para centralizar a tela selecionada aperte Windows + seta para cima.

1. Para maximizar uma tela aperte Windows + M.

1. Para minimizar uma tela aperte Windows + seta para baixo.

1. Para achar uma tela aperte Windows + tab e escolha a tela desejada.

1. Para bloquear seu computador aperte Ctrl + Alt + K, respectivamente para desbloquear digite sua senha do computador, se estiver vermelho a senha foi digitada incorretamente.

# Apps do sistema

1. Para acessar a wifi abra o terminator e digite nmcli device wifi list , escolha sua rede de escolha e digite nmcli device wifi connect NOME_DA_SUA_REDE password SENHA_DA_SUA_REDE .

1. Para escolher as telas a serem usadas abra o monitor settings.

1. Para conectar via bluetooth abra o Bluetooth manager.

1. Para controlar algo relacionado a entrada e saida de som abra o PulseAudio Volume Control.
